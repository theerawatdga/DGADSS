มาตรฐานบริการดิจิทัลภาครัฐ

มสพร. 16-2569 (DGA 16-2569)

ประเภทแอปพลิเคชัน (Application Type)

🌐
1. Website / Web Application
เน้นการให้ข้อมูล ประชาสัมพันธ์ รองรับ SEO มีระบบล็อกอิน
📱
2. Mobile Application
ใช้งานบนสมาร์ทโฟน ใช้ฟีเจอร์เครื่อง (กล้อง, GPS, Push Notification)

ประเภทบริการ (Service Type)

1. บริการข้อมูลข่าวสาร (Informational Service)
2. บริการธุรกรรมกึ่งดิจิทัล (Partial Digital Transactional Service)
3. บริการธุรกรรมดิจิทัลเบ็ดเสร็จ (Fully Digital Transactional Service)
4. พอร์ทัลกลาง (Portal)

มาตรการควบคุม (Controls)

เลือกประเภทแอปพลิเคชันและประเภทบริการเพื่อกรองมาตรการที่เกี่ยวข้อง

Control ID หมวดหมู่ หัวข้อ Compliance รายละเอียด
จำนวนมาตรการที่พบ: 0

การประเมินระดับผลกระทบ (Impact Level)

เลือกระดับผลกระทบในแต่ละด้าน (ตามประกาศคณะกรรมการการรักษาความมั่นคงปลอดภัยไซเบอร์แห่งชาติ พ.ศ. 2566)
